# AproMax Engineering LLP - Website Style Guide

## Brand Overview
AproMax Engineering LLP is a professional engineering firm that requires a website that conveys trust, expertise, and modern technical capability. The design should appeal to B2B clients, government contracts, and industrial partners.

---

## Color Palette

### Primary Colors
- **Primary Blue**: `#0066FF` or `#0052CC`
  - Use for: CTAs, links, active states, key highlights
  - Represents: Trust, professionalism, technology
  
- **Black**: `#1a1a1a` or `#000000`
  - Use for: Headings, primary text, logo text
  - Represents: Authority, precision, sophistication

- **White**: `#FFFFFF`
  - Use for: Backgrounds, cards, contrast areas
  - Represents: Clarity, cleanliness, space

### Secondary Colors
- **Dark Gray**: `#4a4a4a`
  - Use for: Body text, secondary information
  
- **Light Gray**: `#f5f5f5` or `#f8f9fa`
  - Use for: Background sections, alternating content blocks
  
- **Border Gray**: `#e0e0e0`
  - Use for: Subtle dividers, card borders

### Accent Colors (Optional)
- **Success Green**: `#10b981` - for success messages, completed projects
- **Warning Orange**: `#f59e0b` - for important notices
- **Deep Blue**: `#003d99` - for hover states on primary blue

## Typography

### Font Families

**Primary Font (Headings & UI)**
- **Font**: Inter, Roboto, or Source Sans Pro
- **Backup**: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
- **Characteristics**: Clean, geometric, modern, highly readable

**Secondary Font (Body Text)**
- **Font**: Open Sans, Lato, or system fonts
- **Backup**: -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif

### Font Sizes & Hierarchy

```
Hero Heading (H1):        48-64px (3rem-4rem) | Bold | Line-height: 1.1
Page Heading (H1):        36-48px (2.25rem-3rem) | Bold | Line-height: 1.2
Section Heading (H2):     30-36px (1.875rem-2.25rem) | Semi-bold | Line-height: 1.3
Subsection Heading (H3):  24-30px (1.5rem-1.875rem) | Semi-bold | Line-height: 1.4
Card/Component Title (H4): 18-24px (1.125rem-1.5rem) | Semi-bold | Line-height: 1.5
Body Text:                16-18px (1rem-1.125rem) | Regular | Line-height: 1.6
Small Text:               14px (0.875rem) | Regular | Line-height: 1.5
Caption/Label:            12-14px (0.75rem-0.875rem) | Medium | Line-height: 1.4
```

### Font Weights
- Regular: 400 (body text)
- Medium: 500 (labels, small headings)
- Semi-bold: 600 (subheadings, emphasis)
- Bold: 700 (main headings, CTAs)

## Design Style: Modern Professional Industrial

### Core Principles

1. **Clean & Structured**: Grid-based layouts that reflect engineering precision
2. **Professional**: No playful elements, maintain corporate credibility
3. **Modern**: Contemporary design without being trendy or gimmicky
4. **Technical**: Subtle industrial/engineering visual cues
5. **Trustworthy**: Clear hierarchy, easy navigation, transparent information

### Visual Elements

#### Layout
- **Max Width**: 1280px-1440px for content
- **Spacing System**: 8px base unit (8, 16, 24, 32, 48, 64, 96, 128px)
- **Grid**: 12-column responsive grid
- **Margins**: 
  - Desktop: 80-120px side margins
  - Tablet: 40-60px
  - Mobile: 20-24px

#### Cards & Containers
- **Background**: White cards on light gray backgrounds
- **Border Radius**: 4-8px (subtle, not overly rounded)
- **Shadows**: 
  - Default: `0 2px 8px rgba(0,0,0,0.08)`
  - Hover: `0 4px 16px rgba(0,0,0,0.12)`
  - Elevated: `0 8px 24px rgba(0,0,0,0.15)`
- **Borders**: 1px solid #e0e0e0 (when needed)

#### Buttons

**Primary Button**
```
Background: #0066FF
Text: White
Padding: 12px 32px
Font: 16px, Semi-bold
Border-radius: 4px
Hover: #0052CC or slight shadow increase
Transition: 200ms ease
```

**Secondary Button**
```
Background: Transparent
Border: 2px solid #0066FF
Text: #0066FF
Padding: 10px 30px
Hover: Background #0066FF, Text White
```

**Text Button**
```
Background: Transparent
Text: #0066FF
Underline on hover
```

#### Images & Media
- High-quality, professional photography
- Images of: facilities, machinery, team, projects, technical drawings
- Aspect ratios: 16:9 for hero images, 4:3 for project cards, 1:1 for team photos
- Overlay gradients when text is placed on images: `linear-gradient(rgba(0,0,0,0.4), rgba(0,0,0,0.6))`

#### Icons
- **Style**: Outline/stroke icons (not filled)
- **Size**: 20-24px standard, 32-48px for features
- **Color**: Match text color or primary blue
- **Libraries**: Lucide, Feather, or Material Icons Outlined

## Component Styles

### Navigation Bar
- Height: 80px (desktop), 64px (mobile)
- Background: White with subtle shadow
- Sticky/Fixed on scroll
- Logo: Left-aligned
- Menu: Right-aligned, horizontal
- Mobile: Hamburger menu

### Hero Section
- Full-width background image or gradient
- Height: 60-70vh
- Overlay: Dark gradient for text contrast
- Large headline + subheadline + CTA button
- Optional: Animated subtle grid pattern or blueprint lines

### Service/Feature Cards
- White background
- Icon at top (primary blue)
- Title (H4)
- 2-3 sentence description
- Optional "Learn more" link
- Hover: Slight lift effect with shadow

### Project Showcase
- Grid layout: 3 columns (desktop), 2 (tablet), 1 (mobile)
- Image with overlay on hover
- Project title and category
- Click to view case study

### Statistics/Numbers Section
- Dark blue or black background
- White text
- Large numbers with labels
- Example: "500+ Projects Completed"

### Footer
- Dark background (#1a1a1a or #2d2d2d)
- Three columns: About, Quick Links, Contact
- Social media icons
- Copyright info
- White/light gray text

## Interactions & Animations

### Principles
- **Subtle**: No flashy animations
- **Purposeful**: Animations guide attention or provide feedback
- **Fast**: 200-300ms duration
- **Smooth**: Ease-in-out curves

### Specific Animations
- **Page Load**: Fade-in content sections (staggered)
- **Hover States**: 
  - Buttons: Color change + slight scale (1.02)
  - Cards: Lift effect (translate Y -4px) + shadow increase
  - Links: Underline animation or color change
- **Scroll Animations**: Fade-in or slide-up as elements enter viewport
- **Navigation**: Smooth scroll to sections

## Imagery Guidelines

### Photography Style
- Professional, high-resolution images
- Well-lit, clear composition
- Real projects and facilities (avoid generic stock photos)
- Show: machinery, construction sites, engineering drawings, team at work
- Color grading: Slight blue tint to match brand

### Image Treatments
- Subtle overlay gradients when text overlay is needed
- Sharp focus, professional quality

## Responsive Design

### Breakpoints
```
Mobile: 0-640px
Tablet: 641-1024px
Desktop: 1025-1440px
Large Desktop: 1441px+
```

### Mobile Considerations
- Hamburger menu
- Stacked layouts (single column)
- Larger touch targets (minimum 44x44px)
- Simplified navigation
- Compressed spacing (but still generous)

---

## Accessibility

- **Contrast Ratios**: Minimum 4.5:1 for body text, 3:1 for large text
- **Focus States**: Clear visible outline on interactive elements
- **Alt Text**: All images have descriptive alt text
- **Keyboard Navigation**: Full site navigable via keyboard
- **Semantic HTML**: Proper heading hierarchy, ARIA labels where needed

## Do's and Don'ts

### ✅ Do
- Use plenty of white space
- Maintain consistent spacing throughout
- Keep navigation simple and intuitive
- Use high-quality, relevant imagery
- Emphasize credibility (certifications, projects, experience)
- Make contact information easily accessible
- Optimize for mobile devices

### ❌ Don't
- Use overly decorative fonts
- Overuse animations or effects
- Clutter pages with too much information
- Use low-quality or generic stock photos
- Ignore loading performance
- Make forms too long or complicated
- Use trendy design elements that will date quickly

---

## Technical Specifications

### Performance
- Page load time: < 3 seconds
- Image optimization: WebP format with fallbacks
- Lazy loading for images below the fold
- Minified CSS/JS

### SEO
- Semantic HTML5 structure
- Meta descriptions for all pages
- Proper heading hierarchy
- Schema markup for business information
- Mobile-friendly (Google mobile-first indexing)

### Browser Support
- Chrome, Firefox, Safari, Edge (latest 2 versions)
- iOS Safari, Chrome Mobile
- Graceful degradation for older browsers
